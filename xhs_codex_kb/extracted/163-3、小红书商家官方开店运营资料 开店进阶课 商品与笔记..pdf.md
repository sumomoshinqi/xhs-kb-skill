## Page 2

概述
小红书是“社区+电商”模式，为提高转化率，我们鼓励商家进行以下的操作：
【笔记绑商品】在社区热门笔记里，与相关的在售商品进行绑定。

## Page 3

笔记绑商品：第一种，新建笔记直接绑商品
发笔记时，绑定在售商品步骤：
1. 撰写笔记文字内容时，点击“标签”，进入搜索界面；
2. 选择“商品”标签后，在搜索栏输入商品关键字，弹出相关商品列表；
3. 搜索结果列表中，选择商品点击绑定；
4. 绑定成功，发布笔记。

## Page 4

笔记绑商品：第二种，明星笔记绑商品
明星笔记和热门节目等笔记，目前只能走线下流程申请绑定：
1. 找品类对接AM申请；
2. 提供笔记ID和商品ID；
3. 须提供与该明星合作关系的合作协议或者合同，且合作协议和合同上必须
要写清楚合作明星的全名；
4. 品类审核通过后，平台会操作绑定。
