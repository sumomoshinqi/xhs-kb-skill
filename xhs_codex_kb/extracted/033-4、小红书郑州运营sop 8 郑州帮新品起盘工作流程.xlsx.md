# Sheet: Sheet1

阶段 | 关键动作 | 操作细则
1. 选品 | 高毛利品类优先 | • 主流类目：3C/护肤/母婴/清洁/彩妆
 |  | • 高风险类目（谨慎！）：黑五类（增高/减肥/补肾等），毛利80%-3000%+，合规风险极高
2. 上架准备 | 产品包装+基础数据 | • 竞品价析/SKU策划
 |  | • 卖点提炼（差异化成分/功效）
 |  | • 详情页制作
 |  | • 刷评：低客单价30组图评+20视频评
3. 产品分析 | 四维定位 | • 价格带（低/中/高）
 |  | • 用户画像（25-35F/学生/宝妈）
 |  | • 核心卖点（性能/成分）
 |  | • 功效定位（钙片→长高）
4. 竞品解剖 | 对标4-6个竞品 | • 分析其卖点/营销模式（KFS/郑州帮/厦门帮）
 |  | • 监控内容频率/GMV
 |  | • SWOT提炼差异化路径
5. 推广启动 | 账号矩阵+内容铺设 | • 企业号+专业号双开
 |  | • 素人水下笔记+KOC图文直发
 |  | • 配置：2-3人/品，1企号号+1员工号/人
6. 内容测试期 | 多版型AB测试（10篇/型） | • 方向：测评/种草/单推/教程/视频
 |  | • 数据关注：点击率＞转化率＞互动率
 |  | • 爆文要素沉淀：标题钩子/封面关键词/文案框架
7. 流量放大 | 付费工具组合 | • 薯条加热：投自然流量高的优质笔记
 |  | • 信息流：按兴趣标签精准推送
 |  | • 搜索广告：卡位高意图关键词
8. 私域沉淀 | 长效流量转化 | • 微信社群/企微承接
 |  | • 推送复购活动+专属福利
关键风险提示
⚠️ "黑五类"选品：
· 本质：利用焦虑贩卖无效产品（如"5元成本定399"+"无效退款"策略）
· 风险：三无产品、资质缺失、宣传违规，面临平台封禁+法律追责
转型建议：逐步转向合规品类（如护肤/母婴）+ 品牌化（案例：海洋至尊/OIU）
